# 🌿 Canopees – Frontend

Frontend du site vitrine **Canopees**, une entreprise spécialisée dans la conception et l’entretien d’espaces verts.

Ce projet a été réalisé dans le cadre d’une **mise en situation professionnelle** durant la formation **Développeur Web et Web Mobile (RNCP37674)**.

## 🚀 Démo

Application en ligne :
https://yohanfrutiger.github.io/canopees-frontend

Backend API :
https://yohanfrutiger.alwaysdata.net

---

# 🛠 Technologies utilisées

* React
* JavaScript
* Tailwind CSS

---

# 📋 Fonctionnalités

Le site propose plusieurs pages :

### Page d'accueil

* Slider avec photos
* Présentation de l'entreprise
* Galerie des réalisations

### Qui sommes-nous

* Présentation de l’entreprise
* Profil des fondateurs

### Prestations

* Liste des services proposés
* Modales affichant des réalisations

### Tarifs

* Présentation des tarifs

### Contact

* Formulaire de contact
* Demande de devis

---

# 🔐 Back-office

Un espace d'administration permet de gérer :

* les images du slider
* les contenus des pages
* les prestations
* les tarifs
* les messages envoyés via le formulaire de contact

---

# ⚙ Installation du projet

Clone du repository :

```bash
git clone https://github.com/yohanfrutiger/canopees-frontend.git
```

Installation des dépendances :

```bash
npm install
```

Lancer le projet :

```bash
npm start
```

L'application sera accessible sur :

```
http://localhost:3000
```
